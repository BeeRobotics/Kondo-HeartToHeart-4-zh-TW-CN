## 下載和安裝方法

1. 點擊右側 Release 或點擊進入至發行頁面
2. 選擇最新版的 `中文翻譯包.zip` 下載
3. 解壓縮至 `C:\\Program Files (x86)\HeartToHeart4(V2.4.0)`
   把繁體中文包資料夾內的檔案拖進去
4. 開啟 `C:\\Program Files (x86)\HeartToHeart4(V2.4.0)\HTH4.XML`
5. 將最底下的 `<Language>ja-JP</Language>` 或 `<Language>en-US</Language>` 改成 `<Language>zh-TW</Language>`
6. 之後打開軟體就是中文了
